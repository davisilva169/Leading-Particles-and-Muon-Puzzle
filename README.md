# Leading Particles and the Muon Puzzle

This repository hosts the research and computational tools focused on investigating the **Muon Puzzle** in ultra-high-energy cosmic ray (UHECR) physics. The project explores the influence of **leading particles** and hadronic interaction properties on muon production within extensive air showers (EAS).

## 🌌 Overview

The **Muon Puzzle** refers to the persistent discrepancy between the number of muons detected in experimental data (from observatories like Pierre Auger) and the predictions provided by post-LHC hadronic interaction models (such as EPOS-LHC, QGSJET-II-04, and SIBYLL).

This project specifically examines:
* The role of **leading particle energy fractions** in air shower development.
* The impact of the $R$ ratio (the energy ratio between neutral pions and other hadrons).
* Sensitivity of muon multiplicity at ground level to changes in early-stage hadronic interactions.

## 🚀 Features

* **Simulation Processing:** Tools to analyze output from air shower simulators (e.g., CORSIKA).
* **Data Analysis:** Python-based scripts for calculating muon density and $X_{max}$ distributions.
* **Theoretical Modeling:** Implementations of modified hadronic interaction parameters to test potential solutions to the Muon Enigma.
